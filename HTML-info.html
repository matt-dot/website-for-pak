HTML in A level Computer Science

Learners are expected to have an awareness of the following tags:
  • <html>
  • <link> to a CSS file
  • <head>
  • <title>
  • <body>
  • <h1>
  • <h2>
  • <h3>
  • <img> which includes src, alt, height and width attributes
  • <a> including the href attribute
  • <div>
  • <form>
  • <input> where the input is a textbox (has attribute type = "text" and other identification attribute) or has a submit button (has attribute type = "submit")
  • <p>
  • <li>
  • <ol>
  • <ul>
  • <script>
