CSS in A level Computer Science

Learners are expected to be able to use CSS directly inside elements using the style attribute <h1 style=”color:blue;”>

This is known as inline styling

Learners are expected to be able to use external style sheets as well as means to define styling of elements

Examples:
  • Headings and color -
    h1{
    color:blue;
    }
  • Classes -
    .infoBox{
    background-color: #A2441B
    }
  • Identifiers -
    #menu{
    background-color: green;
    }
Learners are expected to be familiar with the following properties:
  • background-color
  • border-color
  • border-style
  • border-width
  • color #A2441B (with both named colors and hexadecimal colors)
  • font-family
  • font-size
  • height
  • width
